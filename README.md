Solved SafeDep Assignment

## Tech Stack Used

- Next.js
- Typescript
- Motion
- TailwindCss

## How to run it locally ?

- Clone the repo

```bash
git clone https://github.com/daman599/SafeDep-assignment.git
```

- Install all the dependencies
- Get your TenantID and ApiKey from https://app.safedep.io 
- Add these into .env file

```bash 
SAFEDEP_API_KEY = Your_API_KEY
SAFEDEP_TENANT_ID = Your_Tenant_ID
```
- Run the development server
- Change your route like this https://localhost:3000/p/{ecosystem}/{name}/{version}
- And there you go.
